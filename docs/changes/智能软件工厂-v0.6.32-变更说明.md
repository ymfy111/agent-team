# 智能软件工厂 v0.6.32 变更说明

## 3. 关键结论

- P0a 是前端可演示 MVP，不接真实后端、数据库、Gateway 或 OpenCode。
- P0a 唯一主线：`Skill → AgentTemplate → Worker → Team/Project → Doc 发布 → Decision 处理 → Activity 回写`。
- `v0.6.32` 原型作为当前 P0a 闭环演示参照，不改变版本号。
- P0b 才引入 Fastify + Prisma + SQLite。
- P1 再完整接入 agent-web-kit + agent-gateway + OpenCode，并引入 PostgreSQL、权限、审计和多用户。
- `chat-integration` 只做适配和转译，不做业务决策。
- `agent-web-kit` 不是业务状态真源。

## 4. 未升版本说明

本次更新虽然补充了较多技术落地内容，但没有改变产品信息架构、核心业务对象、MVP 主目标和原型基线，因此仍保持 v0.6.32。

# 智能软件工厂 v0.6.32 变更说明

> 日期：2026-05-14  
> 基线：v0.6.31 运行态 / 软件工厂配置导航版  
> 本轮目标：完成界面走查后的轻量优化，并补齐开发任务拆解输入。

## 本版更新摘要

1. **导航顺序优化**：软件工厂配置区调整为“岗位 / 员工 / 技能 / 设置”，强化先定义数字员工模板，再创建数字员工实例的心智。
2. **分组文案优化**：左侧导航分组从“运行 / 配置”调整为“运行态 / 工厂配置”。
3. **岗位页增强**：新增主链路提示：岗位定义能力模板 → 创建数字员工实例 → 继承技能 → 匹配团队与项目运行。
4. **技能页增强**：新增 Skill 工作台说明，强调 `SKILL.md`、保存草稿、发布生效和影响范围。
5. **文档同步**：PRD、设计文档、开发计划均同步到 v0.6.32。
6. **新增交付物**：新增《界面走查报告》和《开发任务拆解表》，便于进入 P0a 开发。

## 兼容说明

- 图片引用路径保持不变；本轮不处理原型图片资源缺失问题。
- “岗位”继续作为 UI 菜单短名，技术模型仍为 `AgentTemplate`。
- “员工”继续表示数字员工实例，技术模型可用 `Worker` / `AgentInstance`。

## 历史摘要

- v0.6.31：按运行态 / 软件工厂配置拆分导航，员工 / 岗位 / 技能 / 设置放入配置区。
- v0.6.30：术语收敛为 AgentTemplate / 数字员工模板。
- v0.6.29：引入 Skill 配置工作台与 SKILL.md 编辑形态。
- v0.6.28：技能管理从设置页拆出，设置页恢复只读。

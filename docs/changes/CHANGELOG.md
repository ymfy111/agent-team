# 版本变更说明

> 记录 agent-team 项目各版本的功能变更、改进和修复。

## v0.6.27-skill (2026-05-15)

### 新增
- **技能管理体系**：设置页新增「技能管理」模块，包含技能资产表格和岗位技能匹配视图
- **右滑抽屉编辑器**：支持新建/编辑技能（名称、描述、适用岗位、指令正文、状态），保存时提示影响范围
- **员工卡技能摘要**：每张员工卡底部显示"🔧 N 项技能"
- **员工页统计条**：顶部蓝色统计条，展示总数/在线/忙碌/技能总数
- **10 项 mock 技能数据**：4 项通用技能 + 6 项岗位专属技能

### 数据模型
- Skill 作为独立可复用资产，通过 `applicableRoles` 字段实现多对多关联
- `applicableRoles: ['*']` 表示通用技能，否则为岗位列表

## v0.6.26-persona2 (2026-05-13)

### 新增
- **数字员工人物感**：员工卡增加个性化头像（default/busy/offline 三态）
- **角色徽章**：岗位标签彩色展示
- **协作全景**：总览页团队卡片化布局
- **待决策面板**：支持多选项决策流
- **项目管理页**：项目列表与文档管理
- **团队管理页**：团队创建、成员分配

- V2 注册中心基础扩展原型
- 员工池管理（按团队/按状态分组）

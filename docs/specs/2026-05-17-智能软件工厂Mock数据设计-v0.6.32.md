# 智能软件工厂 Mock 数据设计

> 版本：v0.6.32  
> 日期：2026-05-17  
> 状态：P0a mock seed 正本 / AI 原生岗位版

---

## 1. 设计目标

P0a 的 mock seed 是前端演示和后续工程骨架的唯一初始数据源。mock 数据必须体现当前核心产品理念：

```text
AI 原生岗位
Leader 协调
任务单派发
进度跟踪
阻塞/分歧初判
人类决策
交付审查
整改任务单
Activity 回写
```

P0a 不再使用传统“架构 / 开发 / 测试 / 设计 / 建模 / 文档”作为主岗位心智，而采用：

```text
协同规划岗
实现验证岗
交付审查岗
```

## 3. AgentTemplate / AI 原生岗位模板

### 3.1 模板数量

P0a 当前只保留 3 个核心岗位模板：

```ts

  {
    id: 'tpl-collaboration-planner',

    defaultWorkModes: ['requirements_clarification', 'system_design', 'planning', 'architecture_review'],
```

Worker 命名建议：

```text

```

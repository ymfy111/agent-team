# 原型目录

> 版本：v0.6.33

v0.6.33 起，当前 docs 正本包不再保留多代 HTML 原型文件。

原因：

```text
- 多代 HTML 原型会让 OpenCode 难以判断当前实现入口。
- P0a 已进入施工图阶段，开发应以 specs/ 和 tasks/ 为准。
- 历史原型可通过 Git 历史恢复。
```

如需要视觉对照，可从清空前备份目录或 Git 历史恢复：

```text
docs/prototypes/Agent-Team-V2-注册中心-v0.6.32.html
```

不要把 v0.6.25、v0.6.26、v0.6.27、v0.6.29 等历史原型重新放回当前 docs 正本。

# 智能软件工厂实施路线图

> 版本：v0.6.33  
> 日期：2026-05-17  
> 状态：当前路线图

---

## 1. 阶段总览

```text
P0a：前端可演示 MVP
P0b：本地可运行 MVP
P1：服务化和真实集成
P2：多团队、多项目、多运行时增强
```

---

## 2. P0a：前端可演示 MVP

目标：无后端、无数据库、无真实 Gateway、无真实 OpenCode runtime，跑通可演示业务闭环。

交付：

```text
- pnpm workspace 工程骨架
- contracts/domain/mock-seed/data-client/state
- Skill 工作台
- AgentTemplate 管理
- Worker 创建/启用/停用模拟
- RuntimeHost / RuntimeBinding 展示
- Team / Project 页面
- 文档草稿/发布模拟
- 决策处理
- Activity 回写
- MockChatAdapter
```

验收闭环：

```text
Skill → AgentTemplate → Worker + RuntimeBinding → Team/Project → Doc 发布 → Decision 处理 → Activity 回写
```

---

## 3. P0b：本地可运行 MVP

目标：引入本地后端和持久化，但仍以单机验证为主。

可能交付：

```text
- Fastify API
- Prisma + SQLite
- Team/Worker/Skill/Doc/Decision/Activity 持久化
- 本地 runtime-service
- OpenCode workspace 初始化
- Worker 启用/停用真实文件操作
- AgentRoute/AgentPresence 持久化
```

仍可后置：

```text
- 真实多用户权限
- 生产审计
- 真实 Gateway Provider
- PostgreSQL
```

---

## 4. P1：服务化和真实集成

目标：接入真实 agent-web-kit Gateway、OpenCode RuntimeHost、Provider 查询和生产级状态流。

交付：

```text
- PostgreSQL
- 认证/权限
- Gateway Provider API
- RuntimeHost heartbeat
- OpenCode supervisor
- ConversationRouteBinding 持久化
- 审计日志
- 真实 SSE / command 链路
```

---

## 5. P2：增强方向

```text
- 多项目模型
- 多团队资源调度
- Worker worktree 并行
- 多模型二次评审
- Skill marketplace
- 安全策略 RuntimePolicy
- 自动生成交接文档与复盘
```
